Java Minimal Template Engine

The public API consists of the central class {@link com.floreysoft.jmte.Engine}, the cmd line tool {@link com.floreysoft.jmte.Tool} and two interfaces for configurable aspects of the engine. One aspect is the parsing inside of special expressions inside the template using interface {@link com.floreysoft.jmte.Lexer} and the other is handling erros using {@link com.floreysoft.jmte.ErrorHandler}.
